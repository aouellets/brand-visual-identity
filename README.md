# Brand & Visual Identity

**Six skills for building brands that look intentional from day one.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

A focused set of skills covering every layer of brand and visual identity work: naming, color, typography, moodboarding, logo briefing, and compositional hierarchy. Built for designers, brand strategists, and founders who need structured, opinionated guidance without guesswork.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/brand-visual-identity](https://skillme.dev/pack/brand-visual-identity) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/brand-visual-identity`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Logo Brief Writer](skills/logo-brief-writer/SKILL.md)** — Writes a sharp, complete creative brief for a logo or brand identity project.
- **[Color Palette Builder](skills/color-palette-builder/SKILL.md)** — Builds an accessible, purposeful brand color palette with usage rules.
- **[Typography System](skills/typography-system/SKILL.md)** — Chooses and pairs typefaces and defines a complete type scale for a brand.
- **[Moodboard Builder](skills/moodboard-builder/SKILL.md)** — Assembles a focused moodboard and art direction brief that communicates a brand feel.
- **[Visual Hierarchy](skills/visual-hierarchy/SKILL.md)** — Applies visual hierarchy principles to lay out any composition.
- **[Brand Naming](skills/brand-naming/SKILL.md)** — Generates and evaluates brand or product names against clear criteria.
- **[Brand Guidelines Doc](skills/brand-guidelines/SKILL.md)** — Writes brand guidelines covering voice, tone, visual principles, and usage examples.
- **[Icon System](skills/icon-system/SKILL.md)** — Designs consistent icon systems — grid, stroke weight, optical sizing, naming conventions.
